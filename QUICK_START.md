# ⚡ MindMatters - One Command Startup

## 🚀 Start Everything

### Windows
```powershell
npm start
```

### Linux/macOS
```bash
npm start
```

## 🛑 Stop Everything

### Windows
```powershell
npm run stop
```

### Linux/macOS
```bash
npm run stop
```

## 📍 Service URLs

After starting, access:
- **Main App**: http://localhost:8080
- **Backend**: http://localhost:4000
- **TARS (AI)**: http://localhost:5173
- **Mood Tracker**: http://localhost:5002
- **Voice Analysis**: http://localhost:5000
